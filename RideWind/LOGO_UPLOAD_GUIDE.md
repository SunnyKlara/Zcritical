# Logo上传功能使用指南

## 功能概述

Logo上传功能允许用户自定义设备开机显示的头像图片。支持从相册选择、拍照或使用预设图库。

## 使用方法

### 1. 进入Logo设置界面
- 连接设备后，在主界面（风速调速界面）**向右滑动**
- 进入Logo设置界面

### 2. 选择图片
点击中间的圆形区域（大加号），弹出选择菜单：
- **从相册选择** - 从手机相册选择图片
- **拍照** - 使用相机拍摄新照片
- **预设图库** - 选择内置的预设图片（开发中）

### 3. 裁剪图片
选择图片后会自动进入裁剪界面：
- 裁剪为正方形（1:1比例）
- 调整裁剪区域选择想要的部分
- 确认后图片会显示为圆形预览

### 4. 上传到设备
- 点击"上传到设备"按钮
- 等待上传完成（显示进度条）
- 上传成功后设备会立即显示新Logo

### 5. 恢复默认
- 点击"恢复默认"按钮可删除自定义Logo
- 设备将恢复显示默认头像

## 技术规格

- **图片尺寸**: 154 x 154 像素
- **颜色格式**: RGB565 (16位色)
- **数据大小**: 47,432 字节
- **传输协议**: 分包传输 + CRC32校验

## 蓝牙协议

### 命令格式

| 命令 | 说明 |
|------|------|
| `GET:LOGO` | 查询是否有自定义Logo |
| `LOGO_START:size:crc32` | 开始传输 |
| `LOGO_DATA:seq:hexdata` | 分包数据 |
| `LOGO_END` | 传输结束 |
| `LOGO_DELETE` | 删除自定义Logo |

### 响应格式

| 响应 | 说明 |
|------|------|
| `LOGO:0` | 无自定义Logo |
| `LOGO:1` | 有自定义Logo |
| `LOGO_ERASING` | 正在擦除Flash |
| `LOGO_READY` | 准备接收数据 |
| `LOGO_ACK:seq` | 确认收到第seq包 |
| `LOGO_NAK:seq` | 请求重传第seq包 |
| `LOGO_OK` | 上传成功 |
| `LOGO_FAIL:reason` | 上传失败 |

## 注意事项

1. 上传过程中请保持蓝牙连接稳定
2. 上传大约需要30-60秒，请耐心等待
3. 如果上传失败，可以重新尝试
4. 图片会自动缩放和裁剪，无需手动调整尺寸
