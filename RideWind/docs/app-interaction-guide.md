# RideWind APP 操作交互逻辑文档

> 本文档记录 APP 主控制页面（DeviceConnectScreen）的所有触碰操作逻辑，为引导系统文字提示提供内容依据。

---

## 一、全局布局结构

主控制页面分为两大区域：
- **上半部分（固定）**：汽车模型图片 + 返回按钮 + 菜单按钮
- **下半部分（可滑动）**：三个模式页面，左右滑动切换

三个模式页面（PageView）：
| 索引 | 模式 | 说明 |
|------|------|------|
| 0 | Dev Test | 开发测试模式（最左侧，仅开发用） |
| 1 | Running Mode | 速度/油门控制（默认页面） |
| 2 | Colorize Mode | LED 颜色控制（右滑进入） |

---

## 二、通用操作（所有模式共享）

### 2.1 模式切换
- **操作**：在下半部分区域 **左右滑动**
- **效果**：切换 Running Mode ↔ Colorize Mode（Dev Test 仅开发用）
- **引导文案建议**：「左右滑动切换控制模式」

### 2.2 汽车图片区域 — 单击
- **操作**：**单击** 上半部分汽车图片区域
- **效果**：切换雾化器开关（开启/关闭）
- **反馈**：震动反馈（mediumImpact）
- **引导文案建议**：「点击汽车图片开关雾化器」

### 2.3 汽车图片区域 — 长按
- **操作**：**长按** 上半部分汽车图片区域
- **效果**：弹出关机/重启对话框（下滑关机，上滑重启）
- **引导文案建议**：「长按汽车图片可关机或重启设备」

### 2.4 返回按钮（左上角）
- **操作**：**点击** 左上角返回箭头
- **效果**：返回设备列表页面（如有自定义颜色会先保存）
- **引导文案建议**：「点击返回设备列表」

### 2.5 菜单按钮（右上角）
- **操作**：**点击** 右上角菜单图标
- **效果**：弹出菜单，包含：
  - **Logo 设置**：进入 Logo 上传界面
  - **移除设备**：移除当前设备
- **引导文案建议**：「点击菜单可设置 Logo 或移除设备」

---

## 三、Running Mode（速度控制模式）

### 3.1 显示调速界面
- **操作**：**单击** Running Mode 下半部分区域
- **效果**：显示速度滚轮调节界面
- **引导文案建议**：「点击进入调速界面」

### 3.2 速度调节滚轮
- **操作**：**上下滑动** 速度数字滚轮
- **效果**：调节速度值（0 ~ 340），实时同步到设备
- **反馈**：每格震动反馈（selectionClick）
- **引导文案建议**：「上下滑动调节速度」

### 3.3 单位切换
- **操作**：**点击** 速度数字旁的单位标签（km/h 或 mph）
- **效果**：切换速度单位（公里/小时 ↔ 英里/小时）
- **引导文案建议**：「点击单位可切换 km/h 和 mph」

### 3.4 油门加速（长按）
- **操作**：**长按** 右下角油门按钮区域
- **效果**：持续加速，松开后自动减速回到原始速度
- **反馈**：引擎音效 + 震动反馈
- **引导文案建议**：「长按油门按钮持续加速，松开自动减速」

### 3.5 紧急停止
- **操作**：**点击** 底部紧急停止按钮区域
- **效果**：速度急速归零，停止引擎音效
- **反馈**：强震动反馈（heavyImpact）
- **引导文案建议**：「点击紧急停止，速度立即归零」

## 四、Colorize Mode（颜色控制模式）

Colorize Mode 有两个子界面：
- **Preset（预设配色）**：12 种颜色预设方案
- **RGB Detail（详细调色）**：LMRB 四区域独立 RGB 调色

### 4.1 颜色预设选择（Preset 界面）

#### 4.1.1 滑动选择预设
- **操作**：**左右滑动** 颜色胶囊条
- **效果**：切换 12 种预设颜色方案，选中的颜色实时同步到设备 LED
- **反馈**：每切换一个预设震动反馈（selectionClick）
- **引导文案建议**：「左右滑动选择预设颜色」

#### 4.1.2 点击选择预设
- **操作**：**点击** 非当前选中的颜色胶囊
- **效果**：动画滚动到该颜色并选中
- **引导文案建议**：（同上，可合并）

#### 4.1.3 开始涂色（转盘动画）
- **操作**：**点击** 底部左侧「开始涂色」按钮区域
- **效果**：启动转盘抽奖式颜色循环动画，再次点击停止
- **反馈**：强震动反馈（heavyImpact）
- **引导文案建议**：「点击底部开始颜色循环动画」

#### 4.1.4 进入详细调色
- **操作**：**点击** 底部右侧调色盘按钮区域
- **效果**：切换到 RGB 详细调色界面
- **反馈**：震动反馈（mediumImpact）
- **引导文案建议**：「点击调色盘进入 RGB 详细调色」

### 4.2 RGB 详细调色（RGB Detail 界面）

#### 4.2.1 选择灯带区域（LMRB）
- **操作**：**点击** L / M / R / B 胶囊
- **效果**：选中对应灯带区域（左侧 / 中间 / 右侧 / 后部），下方 RGB 滑条显示该区域颜色值
- **反馈**：轻震动反馈（lightImpact）
- **引导文案建议**：「点击 L/M/R/B 选择灯带区域」

#### 4.2.2 长按灯带区域 — 打开详细调色面板
- **操作**：**长按** L / M / R / B 胶囊
- **效果**：选中该区域并弹出底部 RGB 调色面板 + 右侧亮度调节条
- **反馈**：中等震动反馈（mediumImpact）
- **引导文案建议**：「长按灯带区域打开详细调色面板」

#### 4.2.3 RGB 滑条调色
- **操作**：**左右拖动** R / G / B 滑条
- **效果**：调节对应颜色通道值（0 ~ 255），实时同步到设备 LED
- **引导文案建议**：「拖动滑条调节 RGB 颜色值」

#### 4.2.4 亮度调节
- **操作**：**上下拖动** 右侧亮度调节条
- **效果**：调节 LED 亮度（0% ~ 100%）
- **引导文案建议**：「上下拖动调节亮度」

#### 4.2.5 关闭详细调色面板
- **操作**：**点击** 面板外的半透明遮罩区域
- **效果**：关闭详细调色面板，返回 LMRB 选择界面
- **引导文案建议**：「点击空白区域关闭调色面板」

#### 4.2.6 流水灯速度控制
- **操作**：**左右拖动** 底部「慢 ← → 快」速度滑条
- **效果**：调节流水灯循环速度，拖动即自动开启流水灯效果
- **引导文案建议**：「拖动速度条控制流水灯快慢」

## 五、关机/重启对话框

通过长按汽车图片触发：

### 5.1 关机
- **操作**：**下滑** 关机滑块
- **效果**：执行软关机（关闭雾化器、风扇、LED、LCD）

### 5.2 重启
- **操作**：**上滑** 重启滑块
- **效果**：执行重启（先关闭所有，延迟后重新开启 LCD 和 LED）
