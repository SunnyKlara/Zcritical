# 17. SEO/内容营销系统

## 是什么
通过搜索引擎优化和内容创作，让潜在用户通过搜索找到产品。包括技术 SEO（网站性能/结构/Schema）、内容策略（关键词研究→文章创作→排名追踪）、外链建设。是长期免费流量的核心来源。

## 为什么需要
- 免费的长期流量来源：一篇好文章可以持续带来流量数年
- 建立品牌权威性：搜索"电动车灯效"第一页有你 = 行业权威
- 降低获客成本：SEO 流量 CAC ≈ ¥0（vs 广告 ¥10-50/用户）
- 用户主动搜索 = 高意向：搜索"骑行灯推荐"的人就是目标用户
- 复利效应：内容越多，域名权重越高，新内容排名越快

## 技术栈选型
| 组件 | 技术 | 说明 |
|------|------|------|
| 官网框架 | Next.js 14 (App Router) | SSG + SSR，SEO 友好 |
| 备选 | Nuxt 3 / Astro | Vue 生态 / 纯静态更快 |
| CMS | Strapi / Contentful | 内容管理，非技术人员可编辑 |
| 轻量方案 | Markdown + MDX | 开发者友好，零成本 |

| 分析 | Google Analytics 4 + Search Console | 流量+排名数据 |

| 结构化数据 | JSON-LD (Schema.org) | 富媒体搜索结果 |

## 实现步骤

### Phase 1：官网搭建（3-4h）
1. Next.js 项目初始化：
   ```
   pages/
   ├─ index.tsx          (首页：产品介绍+CTA)
   ├─ product.tsx        (产品详情页)
   ├─ blog/
   │  ├─ index.tsx       (博客列表)
   │  └─ [slug].tsx      (博客详情)
   ├─ docs/
   │  └─ [slug].tsx      (使用文档)
   └─ download.tsx       (App 下载页)
   ```
2. 性能优化：
   - 图片：next/image 自动 WebP + 懒加载
   - 字体：next/font 本地加载
   - CSS：Tailwind CSS purge 未使用样式
   - 目标：Lighthouse 90+ 分
3. 部署：Vercel 自动部署，自定义域名 + HTTPS

### Phase 2：技术 SEO（2-3h）
4. Meta 标签优化：
   ```html
   <title>RideWind - 智能骑行灯效系统</title>
   <meta name="description" content="基于 ESP32 的智能骑行灯效系统..." />
   <link rel="canonical" href="https://ridewind.app/" />
   ```
5. 结构化数据（JSON-LD）：Product / FAQ / Review Schema
6. 技术清单：Sitemap / robots.txt / canonical / OG tags / 移动适配

### Phase 3：内容策略（2-3h）
7. 关键词研究：
   - 核心词：骑行灯 / 自行车灯 / 电动车装饰灯
   - 长尾词：骑行灯效怎么设置 / 自行车氛围灯 DIY
8. 内容日历：每周 1-2 篇，覆盖不同关键词
9. 内容类型：产品页/教程/对比/技术博客

### Phase 4：博客系统（2h）
10. MDX 博客：Markdown + React 组件 + 代码高亮
11. SEO 优化：URL 结构 / H 标签层级 / 内链策略 / 图片 alt
12. 分类/标签/相关文章推荐

### Phase 5：数据追踪与优化（1-2h）
13. Google Search Console + 百度站长平台
14. 排名追踪：核心关键词每周排名变化
15. 内容优化：排名 5-20 的文章重点优化

## 关键坑点
| 坑 | 后果 | 解法 |
|----|------|------|
| 内容质量低 | 搜索引擎不收录 | 每篇 >1500 字，有深度有价值 |
| 关键词堆砌 | 被搜索引擎惩罚 | 自然写作，密度 1-3% |
| 不更新 | 排名逐渐下降 | 定期更新旧文章 + 持续发新内容 |
| 忽略移动端 | 移动搜索占 70%+ | 响应式设计 + 移动端测试 |
| 加载太慢 | 用户跳出 + 排名下降 | SSG + CDN + 图片优化 |
| 百度 vs Google | 两个搜索引擎规则不同 | 国内重百度，海外重 Google |

## 与 RideWind 的关系
- 当前状态：无官网/博客，搜索引擎上找不到产品信息
- 优先级：P5（产品稳定后，作为长期获客渠道）
- 起步方案：一个简单官网 + 3-5 篇核心文章
- 成本：Vercel 免费 + 域名 ¥50/年 + 时间投入
- 目标：6 个月内核心关键词进入搜索前 3 页
