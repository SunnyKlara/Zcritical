# 18. Web 管理后台

## 是什么
给运营/管理员使用的 Web 端管理系统。管理用户、设备、OTA 发布、数据看板、客服工单。是所有后端服务的"操作面板"。

## 为什么需要
- 后端 API 需要可视化操作界面（不能让运营人员用 curl）
- OTA 发布需要审批流程和灰度控制界面
- 用户/设备管理需要搜索、筛选、批量操作
- 数据看板需要图表展示
- 客服需要查询用户信息和设备状态

## 技术架构
```
┌─────────────────────────────────────────────────┐
│              Web 管理后台                         │
├─────────────────────────────────────────────────┤
│                                                  │
│  [前端]                                          │
│   React/Vue + Ant Design / Element Plus         │
│   ├─ 登录/权限（RBAC）                         │
│   ├─ 用户管理（列表/详情/封禁）               │
│   ├─ 设备管理（状态/配置/远程操作）           │
│   ├─ OTA 管理（上传/灰度/发布/回滚）         │
│   ├─ 数据看板（图表/指标/趋势）              │
│   └─ 系统设置（角色/日志/配置）              │
│       │                                          │
│       ▼                                          │
│  [后端 API]                                      │
│   复用业务后端 + 管理专用接口                   │
│   JWT 认证 + 角色鉴权（admin/operator/viewer） │
│                                                  │
└─────────────────────────────────────────────────┘
```

## 技术栈选型
| 组件 | 技术 | 理由 |
|------|------|------|
| 框架 | React 18 + TypeScript | 生态最大，组件库丰富 |
| UI 库 | Ant Design Pro | 企业级后台标配，开箱即用 |
| 状态管理 | Zustand / React Query | 轻量，适合后台 CRUD |
| 图表 | ECharts / Recharts | 数据可视化 |
| 构建 | Vite | 快速开发体验 |
| 部署 | Nginx 静态托管 / Vercel | SPA 部署简单 |

## 实现步骤

### Phase 1：项目脚手架（1-2h）
1. Ant Design Pro 模板初始化
2. 路由配置 + 权限框架
3. API 请求封装（axios + 拦截器 + token 刷新）

### Phase 2：核心页面（4-6h）
4. 登录页 + JWT 认证
5. 用户管理：列表（分页/搜索/筛选）+ 详情 + 操作（封禁/解封）
6. 设备管理：列表 + 状态展示 + 远程配置 + 批量操作
7. OTA 管理：固件上传 + 版本列表 + 灰度规则配置 + 发布/回滚

### Phase 3：数据看板（2-3h）
8. 概览页：今日活跃用户/设备、新增注册、OTA 覆盖率
9. 趋势图：日活/月活折线图、固件版本分布饼图
10. 实时数据：在线设备数（WebSocket 推送）

### Phase 4：权限与审计（1-2h）
11. RBAC：超级管理员 / 运营 / 客服 / 只读
12. 操作日志：谁在什么时间做了什么操作
13. 敏感操作二次确认（OTA 全量发布、用户封禁）

## 关键坑点
| 坑 | 后果 | 解法 |
|----|------|------|
| 权限粒度不够 | 客服能看到不该看的数据 | 细粒度 RBAC + 数据权限 |
| 大列表卡顿 | 万条数据渲染慢 | 虚拟滚动 + 服务端分页 |
| Token 过期 | 操作到一半被踢出 | 静默刷新 + 操作队列 |
| 误操作 | 批量删除/全量 OTA | 二次确认 + 操作日志 + 可回滚 |
| 多人同时编辑 | 数据覆盖 | 乐观锁 + 冲突提示 |
