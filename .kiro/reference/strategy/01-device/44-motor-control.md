# 44. 电机控制算法

## 是什么

通过 PWM 精确控制直流电机/无刷电机的转速、加速曲线、制动响应。包括开环控制（PWM 占空比直接映射）和闭环控制（PID 反馈调节），以及电机保护逻辑。

## 为什么需要

- 风扇转速需要平滑可控（不是只有开/关）
- 用户体验：加速要渐进（不突兀），减速要自然（不急停）
- 电机保护：过流/过热/堵转检测，防止烧毁
- 噪音控制：PWM 频率选择影响电机啸叫
- 未来扩展：如果做电动滑板/自行车助力，电机控制是核心

## 技术架构

│       │                                          │

│       │                                          │

## 技术栈选型

| 组件 | 技术 | 说明 |
|------|------|------|
| PWM 输出 | ESP-IDF LEDC (25kHz) | 超出人耳范围，无啸叫 |
| 电机驱动 | N-MOSFET + 续流二极管 | 低压直流电机标配 |
| PID 库 | 自实现（~50 行） | 简单场景不需要库 |
| 反馈传感器 | 霍尔传感器 / 光电编码器 | 闭环必需 |
| 电流检测 | 分流电阻 + ADC | 过流保护 |
| 加速曲线 | S 曲线 / 梯形加速 | 平滑启停 |

## 实现步骤

### Phase 1：开环 PWM 控制（1-2h）

1. **PWM 频率选择**
   - 25kHz：超出人耳范围，电机不啸叫
   - 10-bit 分辨率（0-1023），足够精细

2. **加速曲线（S 曲线）**
   - 不直接跳到目标值，而是平滑过渡
   - 每 tick 限制最大变化率
   - 启动时慢→快→慢，停止时快→慢

3. **死区补偿**
   - 电机在低 PWM 时不转（摩擦力 > 驱动力）
   - 检测最小启动 duty（如 15%），低于此值直接输出 0

### Phase 2：PID 闭环控制（3-4h）

4. **PID 控制器实现**
   - P 项：比例响应，误差大则输出大
   - I 项：积分消除稳态误差，带 anti-windup 限幅
   - D 项：微分抑制超调，带低通滤波去噪

5. **RPM 测量（霍尔传感器）**
   - 用 PCNT (Pulse Counter) 外设硬件计数
   - 每 100ms 读取计数值换算 RPM
   - 无需 CPU 参与计数，零开销

6. **PID 调参方法**
   - Ziegler-Nichols 法：先只用 P，增大到振荡，记录临界增益和周期
   - 手动法：P→I→D 逐步加入
   - 串口实时输出数据，Python 绘图观察响应曲线

### Phase 3：电机保护（2h）

7. **过流保护**
   - ADC 读取分流电阻电压换算电流
   - 超过阈值立即停止 + 设置故障标志

8. **堵转检测**
   - PWM > 30% 且 RPM == 0 持续 > 500ms → 堵转
   - 立即停止（堵转时电流极大，会烧电机/MOSFET）

9. **温度保护**
   - NTC 热敏电阻贴在电机/MOSFET 上
   - 分级：警告（降功率）→ 停机（超极限）

### Phase 4：高级控制（2-3h）

10. **梯形加速**
    - 加速段：线性增加 PWM
    - 匀速段：PID 维持目标
    - 减速段：线性降低或主动制动

11. **再生制动（无刷电机）**
    - 减速时电机变发电机，能量回收
    - 需要 H 桥驱动电路支持

## 关键坑点

| 坑 | 后果 | 解法 |
|----|------|------|
| PWM 频率太低 | 电机啸叫刺耳 | 用 20kHz+ 超出人耳范围 |
| PID 积分饱和 | 超调严重，响应迟钝 | Anti-windup 限幅 |
| 死区非线性 | 低速时转/不转交替抖动 | 死区补偿 + 最小输出跳变 |
| 反电动势干扰 | ADC 读数不准 | 在 PWM OFF 相位采样 |
| MOSFET 发热 | 长时间大电流烧毁 | 散热片 + 温度保护 |
| 电机感性负载 | 关断时反向电压尖峰 | 续流二极管必须加 |
| 噪声耦合 | 电机 EMI 干扰 BLE/传感器 | 电源隔离 + 滤波电容 |

## 与 RideWind 的关系

- 当前状态：风扇用简单 PWM 开环控制（LEDC），无反馈
- 已有：`drv_pwm.c` 基础 PWM 输出
- 问题：GPIO 40 PWM 对风扇转速无影响（硬件问题已确认）
- 实际控制：风扇只受 GPIO 10 开关控制（全速/停止）
- 未来：如果换有调速能力的风扇，可以加 PID 闭环
